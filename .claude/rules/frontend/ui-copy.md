# UI Copy Rules

## COPY-001: User-facing copy must match the basis exactly

- User-facing copy must match approved basis text exactly, including spec, Figma, Jira, Confluence, or PR requirement text.
- Do not accept semantic equivalence.
- Do not paraphrase, improve, normalize, or localize copy unless explicitly requested.
- Check spelling, spacing, punctuation, capitalization, line breaks, placeholders, units, numbers, and date/time formats.
- Treat mismatch as a gap.
- Default severity: major.
- Escalate to critical for legal, payment, privacy, consent, security, or compliance text.
