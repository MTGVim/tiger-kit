현재 단계는 실행 진행 중입니다.

- Done 2개가 있어 이미 일부 작업은 완료되었습니다.
- In Progress 0개이므로 현재 진행 중인 작업은 없습니다.
- Ready 3개가 있으므로 다음에 실행할 수 있는 작업이 남아 있습니다.
- Blocked 1개는 의존성이나 결정 대기 등으로 아직 진행할 수 없습니다.

다음 작업은 Ready 상태의 작업 중 우선순위가 가장 높은 1개를 선택해 시작하는 것입니다. 작업을 시작하면 해당 항목을 In Progress로 전환하고, 완료 후 Done으로 갱신한 뒤 검증 결과와 실행 기록을 남기면 됩니다.
