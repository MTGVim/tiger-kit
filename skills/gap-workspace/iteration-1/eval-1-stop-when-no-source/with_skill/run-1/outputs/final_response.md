Gap 분석을 시작하려면 원천 자료가 필요합니다.

현재 요청은 “이 저장소에서 부족한 점을 gap 분석해줘”이지만, 분석 기준이 될 원천 자료나 자료 추출 지시가 제공되지 않았습니다. 따라서 저장소를 임의로 원천 자료로 삼아 분석을 시작하거나 `.gap/` 산출물을 만들 수 없습니다.

다음 중 하나를 제공하거나 선택해 주세요.

1. 이슈 트래커 티켓, 문서, PRD, 디자인, 브리프, 스크린샷, 코드 경로
2. “GitHub issues를 가져와”, “이 PR을 기준으로 분석해”, “README와 현재 구현을 비교해”, “docs/usage.md와 skill 동작을 비교해” 같은 자료 추출 지시
3. 인터뷰 방식으로 요구사항 정리 시작

인터뷰 방식으로 진행하려면 먼저 아래 질문에 답해 주세요.

- 해결하려는 문제는 무엇인가요?
- 원하는 완료 상태는 무엇인가요?
- 반드시 지켜야 할 제약이나 제외할 범위가 있나요?
- 참고해야 할 기존 구현, 문서, 화면, 이슈가 있나요?
