# Discovery candidates

Repo rules may include `AGENTS.md`, nested `AGENTS.md`, `CLAUDE.md`, `CLAUDE.local.md`, `.claude/rules/**/*.md`, `SOUL.md`, and actual repo guidance. Repo skills may live in `.agents/skills/`, `.claude/skills/`, or `.hermes/skills/`. User skills may live under the host's `.agents`, `.claude`, `.codex`, or `.hermes` skill directories. Inspect only paths that exist.
