이 플러그인의 `next` 스킬을 사용합니다.

사용자에게는 한글로 답합니다. TIGAP 산출물과 상태 보고도 한글로 작성합니다. 단, 인용한 원문, 코드, 명령어, 파일 경로, 식별자는 원문 그대로 유지할 수 있습니다.

목표: `.gap/{branch_name}/` 산출물을 읽어 현재 workflow 단계, 브랜치/작업 ID 맥락, 진행률, 다음 추천 명령 또는 행동을 보고합니다.

이 명령은 읽기 전용입니다. 파일, 브랜치, git 상태, 산출물을 만들거나 수정하거나 삭제하지 않습니다.
