이 명령은 아래 계약을 직접 따릅니다.

사용자에게는 한글로 답합니다. 인용한 원문, 코드, 명령어, 파일 경로, 식별자는 원문 그대로 유지할 수 있습니다.

목표: 사용자가 “뭐라고?”, “뭐라는거야?”, “무슨말이죠?”, “뭣”, “그래서 뭐 하라는 거야?”, “이거 무슨 뜻이야?”처럼 반응한 긴 LLM 답변, 애매한 설명, 회의록식 덤프를 다시 길게 요약하지 말고 “쉽게 말하면 무엇인지”와 “무엇을 하면 좋은지”만 짧게 정리합니다.

이 명령은 command-only입니다. 자연어 자동 트리거용 skill을 등록하지 않습니다.

## 동작 규칙

1. 원문보다 훨씬 짧게 씁니다.
2. 원문에 없는 일을 만들어내지 않습니다.
3. 확실한 내용과 추정한 내용을 섞지 않습니다.
4. 원문이 실제로 말하는 뜻, 하던 것, 문제 상황을 합쳐 최대 3줄로 해석합니다.
5. 추천은 사용자가 그대로 말하거나 바로 할 수 있는 형태로 최대 2줄만 제시합니다.
6. 원문이 이미 실행 가능하면 “바로 실행해도 되는 말”이라고 짧게 밝힙니다.
7. 원문에 실질 액션이 없으면 추가 질문이나 확인 문장을 제안합니다.

## 기본 출력

```md
뭣? 쉽게 말하면
...

추천
이렇게 말하세요: ...
```

명시적으로 요청받지 않는 한 코드를 구현하거나 파일을 수정하지 않습니다. 사용자가 실제 구현, 파일 수정, 테스트 실행을 요청하면 먼저 이 해석이 맞는지 짧게 확인합니다.
